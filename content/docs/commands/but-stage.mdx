# Stages a file or hunk to a specific branch.

Stages a file or hunk to a specific branch.

Wrapper for but rub `<file-or-hunk>` `<branch>`.

**Usage:** `but stage <FILE_OR_HUNK> <BRANCH>`

## Arguments

* `<FILE_OR_HUNK>` — File or hunk ID to stage (required)
* `<BRANCH>` — Branch ID to stage to (required)
